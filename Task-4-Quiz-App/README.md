# 🧠 Task 4 - Embedded Learning Hub (Quiz Application)

## 📖 Overview

This project was developed as **Task 4** of my **Front-End Development Internship** at **SaiKet Systems**.

The objective of this project was to build an **interactive Quiz Application** using **HTML5**, **CSS3**, and **JavaScript**. The application follows an **Embedded Systems & IoT Learning Hub** theme, allowing users to select a learning module, answer multiple-choice questions, and instantly view their quiz results with performance feedback.

---

## 🎯 Objective

- Build an interactive quiz application using HTML, CSS, and JavaScript.
- Create multiple learning modules for Embedded Systems topics.
- Implement dynamic question loading.
- Calculate and display quiz scores.
- Review incorrect answers after quiz completion.
- Design a clean, responsive, and user-friendly interface.

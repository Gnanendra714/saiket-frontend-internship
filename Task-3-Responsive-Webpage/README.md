# 📱 Task 3 - Responsive Webpage Layout

## 📖 Overview

This project was developed as **Task 3** of my **Front-End Development Internship** at **SaiKet Systems**.

The objective of this project was to design a fully **responsive webpage layout** using **HTML5** and **CSS3**. The webpage follows an **Embedded Systems & IoT Dashboard** theme, featuring a **Header**, **Sidebar**, **Main Content**, and **Footer** while demonstrating responsive design using **CSS Grid**, **Flexbox**, and **Media Queries**.

---

## 🎯 Objective

- Design a responsive webpage using HTML5 and CSS3.
- Create a structured layout with Header, Sidebar, Main Content, and Footer.
- Implement CSS Grid for desktop layouts.
- Apply Media Queries for mobile responsiveness.
- Build a clean and modern dashboard-style interface.

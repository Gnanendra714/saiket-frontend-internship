# 🛒 Task 2 - Styling a Product Card

## 📖 Overview

This project was developed as **Task 2** of my **Front-End Development Internship** at **SaiKet Systems**.

The objective of this project was to design a responsive and visually appealing **Product Card** using **HTML5** and **CSS3**. The product card showcases an **ESP32 Development Board**, maintaining the **Embedded Systems & IoT** theme throughout my internship portfolio while demonstrating clean UI design, proper alignment, and interactive styling.

---

## 🎯 Objective

- Design a modern product card using HTML5 and CSS3.
- Display product information in a clean and structured layout.
- Apply CSS styling for an attractive user interface.
- Implement hover effects for better user interaction.
- Create a responsive design that works across different screen sizes.
